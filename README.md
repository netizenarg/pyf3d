# pyf3d
Python 3d FPS shooter

## run
in your linux console start command
```./run.sh```

## view
![image](./pyf3d.gif "screen")

## lite description
The project's core philosophy is minimal dependencies and the absence of assets. There are no external model files, textures, or sounds—all resources are generated by program code on the fly.

## gameplay
You run as hero, clear world from mobs and level-up with new skills.
W-A-S-D movings;
V (switch camera mode);
F9 menu settings (full edit in config.json);
F10 compass;
F11 stats panel;
F12 fps.
